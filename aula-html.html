HTML (Hypertext Markup Language)- texto interpretado no navegador HTTP (Hypertext Transfer Protocol) - Protocolo de transferência de texto na rede. Quando tem S [HTTPS] ( S - Security) é o mesmo protocolo criptografado geralmente com uma Chave SSL (Secure Socket Layer). Pode ser escrito em qualquer editor de texto Tags html específica para cada finalidade, as princiapais ( html, head, body, h1 a h6, a, p, img). Tags tem pesos padrões (aparência Visual) Abra um editor qualquer, escreva a seguinte estrutura a seguir, salve com o formato .html e abra em qualquer navegador. Tags HTML possui atributos (definem conteúdo, aspecto visual ou funcionalidade) Os atrubutos para formatação foram abandonados com o advento do CSS CSS ( Cascanding Style Sheets ) Folhas de estilo em Cascata Cascata é uma referência ao comportamento que se sobrepoem a aparência visual de acordo com a ordem da declaração CSS. CSS pode ser in-line ou arquivo externo referênciado na forma de link CSS - facilitanto manutenção, escala aos projetos web. HTML e CSS são padrões mantidos pelo W3C ( World Wide Web Consorcium) HTML5 e CSS3 São os mesmos HTML e CSS, porém com novas tags e atributos que dizem respeito ao poder de interpretação dos Browsers Os browsers são baseados em Engines (Webkit, Gecko, Trident, Presto, KHML, etc), as engines seguem os padrões do W3C para definir como os conteúdos web devem ser renderizados. 052677191279
